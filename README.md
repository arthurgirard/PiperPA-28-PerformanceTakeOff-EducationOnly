# PiperPA-28-PerformanceTakeOff-EducationOnly
A-28-181 Takeoff Performance Calculator

> https://huggingface.co/spaces/agrd/PA-28-Performance

This project provides a Python-based tool to estimate the takeoff performance of a PA-28-181 aircraft based on digitized data from the Pilot's Operating Handbook (POH).

How interpolation can be used to estimate values between charted points.

The creation of a user interface (using Gradio) for a technical model.
